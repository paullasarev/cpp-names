Build instructions.

To build the project you will need 'cmake' crocc-platform build tool (http://www.cmake.org/cmake/resources/software.html).

Step-by-step:
* Download cmake from http://www.cmake.org/cmake/resources/software.html
* Install cmake. In inslall dialoge choose 'Add to path' option.
* change dir to project root and run 'cmake .' command
* cmake will create appropriate build scripts. For example, for Visual Studio this will be 'CppNames.sln' file.
* open this file and build in usual way
* or type 'cmake --build .' to build from command line
